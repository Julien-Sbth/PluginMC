Informations sur le Plugin Minecraft

Inventaire

Cette fonctionnalité permet aux joueurs de stocker et de gérer leurs objets.

Boutique d'objets

Les joueurs peuvent acheter des objets dans cette boutique en échange de ressources ou de devises virtuelles.

Blocs détruits

Cette fonctionnalité enregistre les blocs que les joueurs ont détruits dans le monde de jeu.

Blocs parcourus

Les blocs parcourus correspondent aux zones explorées par les joueurs dans le monde de jeu.

Kills

Cette fonctionnalité enregistre le nombre d'ennemis tués par les joueurs.

Achievements

Les achievements sont des objectifs que les joueurs peuvent atteindre pour gagner des récompenses spéciales.
